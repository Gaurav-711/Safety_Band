The Technology

Intelligent detection
in three steps

A two-model AI pipeline running entirely on the device. No internet required. No data leaves the band.

01 — INPUT
🎙️

Continuous Stress Capture

The band's microphone continuously listens in 3-second sliding windows. Vital signs (heart rate, SpO₂) are sampled every second via optical sensors.

02 — DETECTION
🧠

AI Analysis

Whisper embeddings extract 512-dimensional voice features — language agnostic. A unified classifier determines: SAFE, SCREAM, or HELP. Works in Hindi, Bengali, English or any language.

03 — RESPONSE
🚨

Instant Alert

Alert sent via SMS and app notification to designated contacts with GPS location. Raspberry Pi 4 processes everything locally in under 500ms.

The Models

Built on
deep learning

The unified AI model uses OpenAI's Whisper encoder to extract rich numerical patterns from stress — patterns that capture fear, urgency, and distress regardless of what language is spoken.

Trained on real scream recordings and help phrases in multiple languages. Deployed on Raspberry Pi 4 via ONNX runtime for ultra-low latency inference.

Capabilities

Everything she
needs to feel safe

Built on Raspberry Pi 4. Trained with real data. Designed for real life.

🔊

Stress Detection

CNN model trained on real female scream recordings. Detects distress in under 60ms. Works in any acoustic environment.

🌍

Any Language Help Detection

Whisper encoder extracts language-agnostic features. Detects "bachao", "help", "au secours" — any cry for help in any language.

Heart Rate Monitoring

Continuous optical heart rate sensing. Anomaly detection flags sudden spikes consistent with fight-or-flight response.

SpO₂ Tracking

Blood oxygen saturation monitored in real time. Alerts when levels drop below safe thresholds, indicating physical stress.

GPS Alert System

Instant SMS and app notification to designated contacts with precise GPS coordinates when danger is detected.

🔒

On-Device Processing

All AI runs locally on Raspberry Pi 4. No audio is ever transmitted. No cloud dependency. Works without internet.
